sudo nmap -sV -T5 -p- 10.10.154.97

6498/tcp open ssh OpenSSH 7.6p1 Ubuntu 4ubuntu0.3 (Ubuntu Linux; protocol 2.0)

65524/tcp open http Apache httpd 2.4.43 ((Ubuntu))
